# Task Management Web Application

## Technologies Used
- React
- Node.js
- Express
- MongoDB

## Features
- User registration
- User login/logout
- Task CRUD operations
- Task completion tracking
- MongoDB persistence

## Installation

### Backend
```bash
cd server
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm run dev
```

## Environment Variables

Create a `.env` file inside `/server`:

```env
PORT=5001
MONGODB_URI=your_connection_string
JWT_SECRET=your_secret_key
```

## Running the App

Frontend:
http://localhost:5173

Backend:
http://localhost:5001
